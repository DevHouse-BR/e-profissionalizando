Medidor de Velocidade de Banda

Importante: Existem outros fatores não mensuráveis que podem contribuir para reduzir a velocidade de acesso real de uma conexão Internet. Por exemplo: um alto tráfego na Internet causando o congestionamento dos sites incluídos neste teste; acesso de uma conexão Internet compartilhada; uso deste medidor ao mesmo tempo que se faz o acesso a outro site; equipamento do usuário configurado incorretamente (software e/ou hardware); etc.
 
 

Kbps (kilobits por segundo) é a medida utilizada para a taxa de comunicação (transferência de informação) e não a velocidade de acesso.
KBps (kilobytes por segundo) é a medida utilizada para a taxa de transferência de dados ao realizar um download.
