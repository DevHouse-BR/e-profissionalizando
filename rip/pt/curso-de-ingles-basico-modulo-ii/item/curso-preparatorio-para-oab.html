1. Plug-in do Macromedia Flash Player 9.0 ou superior.
2. Windows XP, Vista ou superior com as atualizações mais recentes instaladas.

2. Memória RAM 512 Mb para WIndows XP, 1Gb para Windows Vista ou superior.
