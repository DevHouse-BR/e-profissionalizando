Autocad 2D

O AutoCAD 20007 é um software gráfico com filosofia totalmente voltada ao design, produtividade e compartilhamento de informações. Tem como básico objetivo permitir que o leitor possa de uma maneira clara e direta usufruir dos recursos 2D do AutoCAD de uma forma prática e eficiente.

40Os Principais tópicos abordados no cursos de Autocad 2D são:

Autocad 2007 2D

Aula 1
Conhecendo a área de trabalho
Ambientar-se com a área de trabalho AutoCad 2007 e seus elementos, conhecer a barra de status, abrir arquivos, criar layers e salvar o trabalho.

Aula 2
Primeiros passos manuseando objetos
Criar linhas, trabalhar com seleção de objetos, duplicar e preencher objetos com hachura.

Aula 3
Ferramentas de edição
Trabalhar com arrays, mirrors, utilizar ferramenta zoom, helix, preenchimento gradiente.

Aula 4
Recortando objetos e trabalhando com retangulos
Utilizar a ferramenta Trim (cortar) e criar retangulos.

Aula 5
Manipulando objetos
Utilizar o comando Streech, rotacionar objetos e trabalhar com polilines.

Aula 6
Comando divide, trabalhar com textos e alinhar.
Utilizar o comando divide, criar textos e alinhar objetos.

Aula 7
Dimension Style Manager
Trabalhar com o Dimension Style Manager

Aula 8
Trabalhando com blocos
Inserir, excluir, duplicar e gerenciar blocos.

Aula 9
Trabalhando com formas geométricas
Criar elipses, polígonos, entre outros objetos. Definir tolerâncias.

Aula 10
Gerenciando as configurações da área de trabalho
Configurar a área de trabalho alterando as propriedades do crosshair e ucs.

Aula 11
Imprimindo no AutoCad
Configurar o documento para plotagem.
