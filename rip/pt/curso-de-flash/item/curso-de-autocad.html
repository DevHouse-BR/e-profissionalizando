Autocad 2D

O AutoCAD 20007 é um software gráfico com filosofia totalmente voltada ao design, produtividade e compartilhamento de informações. Tem como básico objetivo permitir que o leitor possa de uma maneira clara e direta usufruir dos recursos 2D do AutoCAD de uma forma prática e eficiente.

  • Conhecendo a área de trabalho;
  • Manuseando objetos - Primeiros passos;
  • Ferramentas  de edição;
  • Recortando objetos e trabalhando com retângulos;
  • Manipulando objetos;
  • Comando divide, trabalhando com textos e alinhar;
  • Dimension Style Manager;
  • Trabalhando com blocos;
  • Trabalhando com formas geométricas;
  • Gerenciando as configurações da área de trabalho;
  •  Imprimindo no AutoCAD.

Autocad 3D

 O AutoCAD 2007 disponibiliza novas formas de trabalhar, conjugando o desenho 2D com o modelo 3D. Adquirindo este curso você terá acesso a informações, conteúdos e dicas importantíssimas para utilizar de maneira eficiente a mais nova versão do Autocad.

  • Conceitos básicos de 3D;
  • Desenhando peças cilíndricas;
  • Conhecendo mais recursos;
  • Criando arrays em 3D;
  • Praticando em 3D;
  •  Trabalhando com estilos de visualização.

Os Principais tópicos abordados nos cursos de Autocad são:

Autocad 2007 2D

Aula 1
Conhecendo a área de trabalho
Ambientar-se com a área de trabalho AutoCad 2007 e seus elementos, conhecer a barra de status, abrir arquivos, criar layers e salvar o trabalho.

Aula 2
Primeiros passos manuseando objetos
Criar linhas, trabalhar com seleção de objetos, duplicar e preencher objetos com hachura.

Aula 3
Ferramentas de edição
Trabalhar com arrays, mirrors, utilizar ferramenta zoom, helix, preenchimento gradiente.

Aula 4
Recortando objetos e trabalhando com retangulos
Utilizar a ferramenta Trim (cortar) e criar retangulos.

Aula 5
Manipulando objetos
Utilizar o comando Streech, rotacionar objetos e trabalhar com polilines.

Aula 6
Comando divide, trabalhar com textos e alinhar.
Utilizar o comando divide, criar textos e alinhar objetos.

Aula 7
Dimension Style Manager
Trabalhar com o Dimension Style Manager

Aula 8
Trabalhando com blocos
Inserir, excluir, duplicar e gerenciar blocos.

Aula 9
Trabalhando com formas geométricas
Criar elipses, polígonos, entre outros objetos. Definir tolerâncias.

Aula 10
Gerenciando as configurações da área de trabalho
Configurar a área de trabalho alterando as propriedades do crosshair e ucs.

Aula 11
Imprimindo no AutoCad
Configurar o documento para plotagem.

Autocad 2007 3D

Aula 1
Conceitos básicos de 3D
Criar um trabalho simples utilizando o comando extrude(estender um perfil desenhado em poliline). Utilizar o comando subtract para transformação de um sólido em elementos vazados.

Aula 2
Desenhando peças cilindricas
Desenvolver peças em formatos variados e trabalhar com os modos de visualização.

Aula 3
Conhecendo mais recursos.
Criar objetos 3d com os comandos box, cylinder e union.

Aula 4
Criando arrays em 3D
Aprenda a criar arrays em 3D com o comando 3D array e a criar cortes perfeitos com o comando slice.

Aula 5
Praticando em 3D
Criar o projeto de uma residencia simples, incluindo elementos como portas e janelas.

Aula 6
Trabalhar com estilos de visualização.
Conhecer os modos de visualização e acabamento para objetos 3D.
